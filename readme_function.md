# Mini Action Task 功能指引

## 这是什么 App

Mini Action Task 是一个强调“先推进、再优化”的任务管理工具。它把任务拆成可执行的下一步动作，帮助你在低门槛启动后持续推进。

## 核心特点

- 任务分状态管理：待选、进行中、已完成、冻结、删除。
- 下一步动作机制：支持批量输入，每一行代表一个动作，适合把复杂任务拆成小步骤。
- 智能推荐：基于近期能量状态与任务属性推荐可执行任务，减少选择成本。
- 自动冻结保护：超过一定天数后自动冻结未完成的任务，减少任务堆积带来点压力。
- 数据统计视图：包含近7天趋势、近4周完成分布、20周动作热力图。
- 个人中心能力：提醒语录、自定义头像签名、CSV导入导出、数据库备份恢复。

## 推荐使用技巧

- 用“待选”做任务池，用“进行中”控制在手任务数量，避免并行过多。
- 卡住时不要删任务，先冻结并写原因，后续恢复时更容易接上上下文。
- 低状态时优先做低能量动作，保持最低程度的连续性比追求单日爆发更重要。
- 每周导出一次 CSV 或做数据库备份，便于留档与迁移。

## 适合推荐给谁

- 想把拖延任务快速启动的人。
- 习惯把目标拆成行动清单的人。
- 想通过可视化复盘持续优化执行节奏的人。
